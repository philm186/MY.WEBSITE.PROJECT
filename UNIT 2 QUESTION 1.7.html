Describe which are the appropriate file types for websites?

There are many different types of file formats to choose from and some of the more appropriate ones to use for websites i think are JPEG, PNG, SVG AND GIF and all contain a lossy or lossless compression. a lossy compression means images lose information and quality once the file size decreases whereas a lossless compression does not lose any detail when the file size is changed. some other file types are .AVI — Audio/Video Interleaved,CSS — Cascading StyleSheet, DOC — Microsoft Word DOCument and .HTML/ .HTM — HyperText Markup Language file Most pages you create for a website will be HTML files
